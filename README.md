
# 🏦 SmartBank - Python Banking System

A console-based banking application built using **Python** that simulates basic banking operations with secure PIN authentication and persistent data storage using file handling.

## 📌 Project Overview

SmartBank is a beginner-friendly banking system developed to strengthen my understanding of Python fundamentals, Object-Oriented Programming (OOP), and file handling. The application stores user data in text files so that account information persists even after the program is closed.

This project was built incrementally, with each version introducing new features and improvements.

---

## ✨ Features

* 💰 Check Account Balance
* ➕ Deposit Money
* ➖ Withdraw Money
* 📜 View Transaction History
* 🔐 Change Account PIN
* 🔑 PIN Verification with Limited Attempts
* 💾 Persistent Data Storage using File Handling
* 👤 Multiple User Support
* 🕒 Transaction History with Timestamps
* 📈 Interest Calculation
* ⚠️ Input Validation and Error Handling
